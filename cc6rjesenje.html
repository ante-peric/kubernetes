Riješenje zadatka 6 Core Concept:

6. Zadatak

$ vim s_pod.yml

apiVersion: v1
kind: Pod
metadata:
  name: ante
  labels:
    app: ante-aplikacija
    tier: front-end
    author: Ante-Peric

spec:
  containers:
    - name: nginx-conatainer
      image: nginx

$ kubectl create -f s_pod.yml

6.1. Zadatak

$ vim ante_service_pod.yml

apiVersion: v1
kind: Service
metadata:
  name: ante-aplikacija-service
  labels:
    app: ante-aplikacija
    tier: front-end
    author: Ante-Peric

spec:
  type: NodePort
  ports:
    - targetPort: 80
      port: 80
      nodePort: 30009

  selector:
    app: ante-aplikacija
    tier: front-end
    author: Ante-Peric

$ kubectl create -f ante_service_pod.yml

6.2. Zadatak

$ kubectl get services
NAME TYPE CLUSTER-IP EXTERNAL-IP PORT(S) AGE
ante-aplikacija-service NodePort 10.109.253.233 80:30009/TCP 13m

$ curl http://10.109.253.233

$ curl http://192.168.122.120:30009
