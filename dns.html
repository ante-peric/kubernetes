2.2. DNS


U ovom tutorialu nećemo koristiti DNS server, umjesto DNS servera koristiti ćemo “hosts” file za određivanje FQDN – a. Na svakoj radnoj stanici /etc/hosts file je uredjen na način:

192.168.122.120 controlkube.caksonic.local controlkube
192.168.122.121 worker1.caksonic.local worker1
192.168.122.122 worker2.caksonic.local worker2
