2.6. Kernel Moduli


Kreiramo I učitavamo nove kernel module: Na svakoj radnoj stanici dodajemo kernel module:

  • overlay
  • br_netfilter


Kreiramo file:

vim /etc/modules-load.d/containerd.conf



Sa sadržajem

overlay
br_netfilter


Učitavamo novokreirane module

modprobe overlay
modprobe br_netfilter


Dodajemo kernel parametre s kreiranjem file – a:

# vim /etc/sysctl.d/k8s.conf


net.bridge.bridge-nf-call-iptables = 1
net.ipv4.ip_forward = 1
net.bridge.bridge-nf-call-ip6tables = 1


Učitavanje novo – kreiranih parametara

# sysctl --system
