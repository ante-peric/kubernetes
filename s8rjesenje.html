8.1. Zadatak

Možemo upotrijebit sljedeće:

kubectl run static-busybox --image=busybox --dry-run=client -o yaml --command -- sleep 1000

Ovaj kod nam kreira predložak i sad možemo kreirati .yml file

kubectl run static-busybox --image=busybox --dry-run=client -o yaml --command -- sleep 1000 > static-busybox.yml

Uređujemo file:

vim static-busybox.yml

apiVersion: v1
kind: Pod
metadata:
  labels:
    run: static-busybox
  name: static-busybox
spec:
  containers:
  - command:
    - sleep
    - "1000"
    image: busybox
    name: static-busybox

mv static-busybox.yml /etc/kubernetes/manifests/
