Zadatak 1:

Kreiraj na svakom radniku direktorij: /mnt/data-volume

Zadatak 1.1.

Kreiraj hostPath Volume unutar pod - a pod1.yml sa sljedećim svojstvima:
pod:
name: random-pod
image: almalinux
command: echo “Hello Ante”

pod volume:
name: data-volume
path: /mnt/

host volume:
name: data-volume
path: /mnt/data-volume
type: Directory

1.2. Zadatak

Napomena: direktorij /mnt/data-volume1 mora biti kreiran na workerima

Kreiraj hostPath Volume unutar pod - a pod2.yml sa sljedećim svojstvima:

pod:
 name: random-pod2
image: almalinux
command: mora ispisivat “Hello Ante” u /mnt/ante.txt

pod volume:
name: data-volume1
path: /mnt/

host volume:
name: data-volume1
path: /mnt/data-volume
type: Directory

1.3. Zadatak

Napomena: direktorij /mnt/data-volume3 mora biti kreiran na workerima

Kreiraj hostPath Volume unutar Deployment - a dep3.yml sa sljedećim svojstvima:

deployment:
 name: random-deployment
pod:
 name: random-pod3
labels:
type: jsp
image: almalinux
replicas:3
selector: sve iz label
command: mora ispisivat “Hello Ante” u /mnt/ante.txt

pod volume:
name: data-volume3
path: /mnt/

host volume:
name: data-volume3
path: /mnt/data-volume3
type: Directory
