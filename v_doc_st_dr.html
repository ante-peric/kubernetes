2. Docker Storage Drivers


Docker pohranjuje podate na lokalnom računalu. Pri instalaciji Dockera on kreira strukturu direktorija koje koristi.

/var/lib/docker

• aufs
• containers
• image
• volumes

Doker nam omogućava I kreiranje direktorija za lokalnu pohranu podataka:

docker volume create data_volume

Direktorij se kreira :

/var/lib/docker/volumes/data_volume

sa naredbom mount ovaj direktorija primjer za sql:

docker run -v data_volume:/var/lib/mysql mysql

Ako želimo podatke van defaultnog Docker directorija na primjer imamo direktorij:

/home/ante/podaci , tada pokrećemo naredbu s punim putem.

docker run -v /home/ante/podaci:/var/lib/mysql mysql

ovo se zove bind mount.

Storage drivers lista:
• AUFS
• ZFS
• BTRFS
• Device Mapper
• Overlay
• Overlay2

Volumes Drives su održavani od strane Volume Drivers Plugins.
